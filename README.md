### 链表
#### 单向链表（linkedlist）
链表node只有element和next，只有从头到尾遍历
#### 双向链表（double)
双向链表实现从尾到头遍历就很简单了，找到尾节点，while循环即可

#### 循环链表
循环脸变和单向链表类似，只是head.next指向head自己，在遍历自己的时候要修改遍历条件，这里就不实现了
